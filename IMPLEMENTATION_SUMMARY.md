- [x] Mobile-first CSS с поддержкой тем
- [x] CSS-токены для light/dark/system тем

### Phase 2 - Formula engine ✅
- [x] `formulas.js` - чистые функции расчёта
- [x] Нормализация процентов (6 -> 0.06, 0.06 -> 0.06)
- [x] Все формулы из RFC:
  - Закупка в ₽ = CNY × курс
  - Карго = вес × ставка × USD/RUB
  - Страховка = закупка × %
  - Себестоимость = сумма всех затрат
  - Цена продажи = себестоимость × (1 + наценка%)
  - Налоги = цена × налог%
  - Прибыль = цена − себестоимость − налоги
  - Маржа = прибыль / цена

```bash

```

Откройте `http://localhost:8080` в браузере.
