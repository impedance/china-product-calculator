# China Product Calculator

Мобильный PWA-калькулятор для расчёта себестоимости, налогов, прибыли и маржи при закупке товаров из Китая.

## Возможности

- **Расчёт себестоимости** с учётом закупки, доставки, карго, страховки, переработки и упаковки
- **Автоматический пересчёт** в реальном времени при изменении данных
- **Пресеты примеров** для быстрого демо (Пример A и Пример B)
- **Сохранение данных** в localStorage с восстановлением при перезагрузке
- **Поддержка офлайн** благодаря Service Worker
- **Тёмная тема** с поддержкой системных настроек
- **Мобильный-first** дизайн, оптимизированный для смартфонов

## Формулы расчёта

1. **Закупка в ₽** = Цена (CNY) × Курс CNY/RUB
2. **Стоимость карго** = Вес × Ставка (USD/кг) × Курс USD/RUB
3. **Страховка** = Закупка × Страховка %
4. **Себестоимость** = Закупка + Доставка по Китаю + Карго + Страховка + Переработка + Упаковка
5. **Цена продажи** = Себестоимость × (1 + Наценка %)
6. **Налоги** = Цена продажи × Налог %
7. **Прибыль** = Цена продажи − Себестоимость − Налоги
8. **Маржа** = Прибыль / Цена продажи

## Установка

### Локальный запуск

```bash
# Простой HTTP-сервер для тестирования
cd /home/spec/work/chi-calc
python3 -m http.server 8080

# Откройте в браузере
http://localhost:8080
```

### Установка как PWA

1. Откройте приложение в Chrome/Safari/Edge на мобильном устройстве
2. Нажмите "Добавить на главный экран" / "Install"
3. Приложение будет работать офлайн

## Тестирование

### Unit-тесты

Откройте `tests.html` в браузере для запуска автоматических тестов.

Тесты покрывают:
- TC-01: Spreadsheet baseline (Example A)
- TC-02: Percent whole-number normalization (100 -> 1, 6 -> 0.06)
- TC-03: Percent decimal normalization (1 -> 1, 0.06 -> 0.06)
- TC-04: Zero tax calculation
- TC-05: Zero markup calculation
- TC-06: Safe division (zero retail)
- TC-07: Cargo dependency check
- TC-08: Total sum integrity
- TC-09: Negative input rejection
- TC-10: Empty required field behavior

### Ручное тестирование (Smoke tests)

1. **Загрузка примера A** - Нажмите "Пример A" и проверьте все KPI-карточки
2. **Изменение наценки** - Измените наценку, проверьте автоматический пересчёт
3. **Кнопка "Рассчитать"** - Очистите форму, нажмите "Рассчитать" - должен появиться чеклист
4. **Сохранение** - Нажмите "Сохранить", обновите страницу - данные должны восстановиться
5. **Тёмная тема** - Переключите тему, обновите страницу - тема должна сохраниться
6. **Офлайн** - Отключите интернет, обновите страницу - приложение должно работать
